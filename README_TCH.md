## Update更新

### ver 1.0:

- 包含 5 個武器 和 2 個魔法。 5 種怪物。 和 讀取地圖的系統。
- 血條、魔力條、和升級系統。

```
方向鍵移動
Q 切換武器
E 切換魔法
空格 使用武器
左_ctrl 使用魔法
M 打開升級選單
P 暫停遊戲
```
